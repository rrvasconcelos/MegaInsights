# Mega Insights

Web scraper para resultados da loteria desenvolvido em .NET 9.

## 🚀 Como começar

### Pré-requisitos
- .NET 9
- SQL Server
- Chrome/Firefox

### Instalação
1. Clone o repositório
   ```bash
   git clone https://github.com/seu-usuario/mega-insights.git
2. Configure as variáveis de ambiente
3. Execute as migrations
4. Rode o projeto
